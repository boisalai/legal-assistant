# Guide de Contribution

## ⚠️ Projet Propriétaire

Ce projet est la propriété exclusive de [Ta Compagnie].

### Accès au code
- L'accès est strictement limité aux collaborateurs autorisés
- Tout contributeur doit signer un NDA avant d'accéder au code
- Le code ne peut être copié ou partagé en dehors du projet

### Process de développement
1. Créer une branche depuis `develop`
2. Faire vos modifications
3. Créer une Pull Request
4. Attendre la revue de code
5. Merger après approbation

### Standards de code
- Python: PEP 8
- TypeScript: ESLint + Prettier
- Tests obligatoires pour toute nouvelle fonctionnalité
